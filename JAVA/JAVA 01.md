### 1. 왜 JAVA를 선택하게 되었어요?

* 객체 지향을 설명하기 좋은 언어이기 때문.
* 자바 플랫폼(JVM)을 통한 확장성.
* GC가 있어서 메모리 다루기가 쉽다.

### 2. JAVA의 특징

* 운영체제 종속적이지 않다.
* 객체 지향 언어이다.
* 안드로이드 개발 가능.
* 국내 JAVA 점유율.
* php(1000건) < jsp(1900건) - 2018-09-25 국내 기준
